# Aws s3 Boto3 Access Datalake

## Ricardo Marques

**Obtendo acesso a um datalake s3 da AWS.**

Objetivo inical fazer a extração e load destes arquivos .parquet encontrados para tabelas em um banco de dados relacional.


**Bibliotecas Utilizadas:**
- Boto3 - Para obter acesso
- Pandas - Transformar os dados em um Dataframe e salvar o arquivo em um banco de dados no formato Tabela.
- sqlalchemy - Conexão com o Banco de dados
